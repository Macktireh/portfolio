Application détection des masques

J’ai réalisé ce projet pour creer une application qui permet de détecter si une porte un masque ou pas.

Ce système peut donc être utilisé dans des applications en temps réel qui nécessitent la détection de masques faciaux à des fins de sécurité en raison de l'épidémie de Covid-19. Ce projet peut être intégré à des systèmes embarqués pour une application dans les aéroports, les gares, les bureaux, les écoles et les lieux publics afin de garantir les gestes barrières.

Technologies utilisées : PYTHON, OPEN-CV, FLASK, HTML, CSS, HEROKU
